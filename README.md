## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/sql-mastery.git
   cd sql-mastery
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173`
